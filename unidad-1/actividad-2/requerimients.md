# Descripción del Proyecto: Calculadora Interactiva con WebComponents

Este proyecto consiste en el desarrollo y migración de una calculadora básica hacia la tecnología de WebComponents, utilizando JavaScript moderno y estándares web. La implementación se basa en la derivación de la clase `HTMLElement`, permitiendo la creación de componentes reutilizables y encapsulados que mejoran la modularidad y mantenibilidad del código.

## Tecnologías Utilizadas
- **Lenguaje**: JavaScript (ES6+)
- **Framework/Abstracción**: WebComponents nativos (sin librerías externas)
- **Estilos**: CSS para diseño responsivo y accesible
- **Herramientas**: Navegadores modernos compatibles con Custom Elements

## Objetivos y Beneficios
- **Funcionalidad Preservada**: La calculadora mantiene operaciones básicas (suma, resta, multiplicación, división) y manejo de decimales, asegurando una experiencia de usuario fluida.
- **Abstracción Avanzada**: Implementa la interfaz mínima de WebComponents, promoviendo buenas prácticas de desarrollo web como encapsulamiento, reutilización y separación de responsabilidades.
- **Aplicabilidad Empresarial**: Demuestra habilidades en desarrollo frontend moderno, ideales para proyectos escalables en entornos corporativos que requieren componentes modulares y compatibles con estándares web.
